## Overview

CareLynk is a modern, comprehensive healthcare management web application designed to streamline healthcare operations. The platform provides healthcare providers with an intuitive interface to manage patients, nurses, sponsors, appointments, and medical records.

## Key Features

✅ **Dashboard** - Real-time statistics, charts, and activity summaries
✅ **User Management** - Complete CRUD operations for Admins, Nurses, Patients, and Sponsors
✅ **Patient Profiles** - Detailed medical records, medication tracking, and sponsor management
✅ **Appointment Scheduling** - Multiple view options (Day, Week, Month) with full appointment management
✅ **Chat System** - Real-time messaging between providers and patients
✅ **Settings** - Comprehensive user preferences and account management
✅ **Notifications** - Notification panel with categorized alerts
✅ **Fully Responsive** - Works seamlessly on desktop, tablet, and mobile devices

## Technology Stack

- **Frontend:** React 18 + TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **UI Components:** Radix UI
- **Charts:** Recharts
- **Routing:** React Router v7
